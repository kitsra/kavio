# Fixed-frame visual fixture

This fixture gives the browser renderer a deterministic set of frame inputs for
future golden-frame and parity tests. It intentionally uses only inline data and
JSON so it can run in Node without Playwright, Chromium, or external assets.

- `composition.json` is the render input.
- `frames.json` lists the fixed frames that should remain stable.
- `browser-renderer.dom-snapshot.json` is generated by
  `packages/browser-renderer/test/visual-fixtures.mjs`.

Regenerate the snapshot after intentional fixture or renderer-output changes:

```sh
corepack pnpm --filter @kavio/browser-renderer test -- --update-visual-fixtures
```
